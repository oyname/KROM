## in development

Main page: https://oyname.github.io/index.html

Die KROM Engine ist eine moderne, datenorientierte C++20-Engine mit klarem Fokus auf Render-Performance und Erweiterbarkeit. 
Die Architektur folgt bewährten Patterns aus der Game-Dev-Praxis (ECS, RenderGraph, Plugin-System).
